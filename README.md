# Capstone Project

 ## Overview
Capstone project for the AI-assisted development track. This repo demonstrates a working developer toolchain — Node.js, Git, and Claude Code — used to scaffold, document, and iterate on a small project following Conventional Commits and AI-assisted workflows.
## Status
🚧 In progress — this is the initial project scaffold.

## Getting Started

### Prerequisites
- Node.js (LTS)
- Git

### Installation
```bash
git clone https://github.com/yourusername/your-repo-name.git
cd your-repo-name
npm install
```

### Running locally
```bash
npm run dev
```
